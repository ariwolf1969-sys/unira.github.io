Completá estos datos para activar la app

Google Maps API Key

Necesitás una clave de API de Google Maps para el funcionamiento completo del mapa, autocomplete de direcciones y rutas reales.

  1. Ir a console.cloud.google.com
  2. Crear un proyecto (o seleccionar uno existente)
  3. Ir a "APIs y servicios" → "Biblioteca"
  4. Habilitar: Maps JavaScript API, Places API, Directions API
  5. Ir a "Credenciales" → "Crear credenciales" → "Clave de API"
  6. Restringir a tu dominio de GitHub Pages
  7. Copiar y pegar la clave abajo

Si no ponés clave, la app funciona con OpenStreetMap (sin autocomplete ni rutas reales).

Google Form (Datos de Choferes)

Los datos de los choferes se envían automáticamente a un Google Form que vos creás. Luego podés verlos en Google Sheets.

  1. Ir a forms.google.com y crear un formulario
  2. Agregar estos campos en orden: Nombre, DNI, Fecha Nac, Email, Tel, Direccion, Localidad, Marca Veh, Modelo Veh, Año, Patente, Color, Tipo Veh, Nro Chasis, Fecha Registro
  3. Hacer clic en "Responder" (vista previa del formulario)
  4. Clic derecho → "Inspeccionar" en cada campo para ver el name="entry.XXXXXXX"
  5. Pegar la URL del formulario abajo

Sin formulario, los datos se guardan localmente en el navegador del chofer.
